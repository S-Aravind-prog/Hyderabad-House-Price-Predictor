# 🏠 Hyderabad House Price Predictor

A Machine Learning web application that predicts house prices in Hyderabad based on:

- 📍 Location
- 🏢 Property Type
- 🏗 Building Status
- 🛏 BHK
- 📐 Area (sq ft)

## Features

✅ House price prediction
✅ Price displayed in Lakhs/Crores
✅ Property summary
✅ House category detection
- Budget House
- Mid Range House
- Luxury House

✅ Clean modern UI

## Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- HTML
- CSS

## Machine Learning

Model:
- Linear Regression

Preprocessing:
- One Hot Encoding
- Standard Scaling
- Outlier Removal (IQR Method)
